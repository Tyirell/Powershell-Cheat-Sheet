# Cmdlets

## Write-Host
Displays output to the Host. Commonly used when trying to create a fancy colored output to the host

### Syntax
`[-Object] <Object>]`  

`[-NoNewline]`  

`[-ForegroundColor <ConsoleColor>]`  

`[-BackgroundColor <ConsoleColor>]`
